About This Project

NordWireConnect is a Python application where you can connect to NordVPN using the official Wireguard app without using the NordVPN app. This may be useful due to startup issues in the NordVPN app and you can't get the most suggested server in the original Wireguard app.

Note: NordWireConnect is an app that is not made by NordVPN themselves and uses an unofficial method of connecting. Using this app will lack features that the main app may have. Please consider using the official app from NordVPN before trying this!

How It Works

First, it installs the official Wireguard app from wireguard.com if it detects not installed. Then, it fetches server and credential information from NordVPN. Next, it generates a Wireguard configuration file and connects using the Wireguard CLI API.
